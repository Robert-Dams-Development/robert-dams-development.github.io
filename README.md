# Damsdevelopment

Static company website for Damsdevelopment, served via GitHub Pages at [damsdevelopment.nl](https://damsdevelopment.nl) (with `damsdevelopment.com` redirecting to `.nl`).

Freelance landing page targeting MKB clients in the Brainport region: process automation, data & dashboards, system integration.

## Stack

Plain HTML, CSS, JS — no build system. Push to `main` deploys.

## Structure

- `index.html` — single-page landing
- `assets/` — styles, scripts, photo, favicons
- `experience/` — employer detail pages (PinkRoccade, Baxter, Koma, ASML)
- `CNAME` — custom domain
- `.nojekyll` — disables Jekyll processing
